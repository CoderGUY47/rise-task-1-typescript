## 📁 Project Setup

### Running this Project
```bash
# Clone the repo
git clone https://github.com/CoderGUY47/rise-task-1-typescript.git
cd rise-task-1-typescript

# Install dependencies
npm install

# Run the TypeScript file
npm run dev
# or
npx ts-node index.ts
```
